# 北极熊24赛季哨兵仓库

## 简介
北极熊的哨兵电控部分代码。

> 注： 本仓库**已停止维护** ，现着力于开发更易于使用的[StandardRobot++通用机器人框架](https://gitee.com/SMBU-POLARBEAR/StandardRobotpp)，及其配套的电控套件。

### 如何使用
左拨杆控制开火，右拨杆控制云台模式。

左摇杆控制云台yaw和pitch，右摇杆控制底盘运行
